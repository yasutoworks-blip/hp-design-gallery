# 職人・工事業専門 ホームページデザイン見本帳

職人・小規模工事業者向けホームページ制作サービスのデザインサンプル集です。
GitHub Pagesで公開しています。

## ディレクトリ構成

```
index.html          デザイン見本帳（トップページ）
demos/              デモサイト本体（1業種1フォルダ）
  01-interior/      内装・クロス職人向け
  02-painting/      塗装店向け
  03-electric/      電気工事店向け
  04-reform/        リフォーム工務店向け
  05-restore/       原状回復・解体業向け
assets/             OGP画像などの共有素材
screenshots/        作業記録用スクリーンショット（リポジトリには含めない）
```

## 注意

デモサイトに登場する会社名・人物・数値・連絡先はすべて架空のものです。
各デモページには noindex を設定しています。
